# Como executar?
Primeiramente precisa clonar o repositório, onde há duas maneiras simples de fazê-lo

 Com Git Bash: 
- Clique no botão "code" em qualquer branch, copie esse endereço
- Em seguida abra a pasta selecionada para conter o projeto
- Com botão direito selecione a opção git bash 
- Digite git clone e cole o endereço ( git clone -https://github.com/rafapiazza/entrevistaIA.git )
- Em seguida dê espaço digite -b e uma branch a sua escolha (  git clone -https://github.com/rafapiazza/entrevistaIA.git -b dev )
- Pronto, repositório clonado

Com GitHub Desktop:
- Clique no botão "code" em qualquer branch
- Selecione o botão "open with github desktop"
- A partir daí siga as instruções do aplicativo
- Não se preocupe, a interface é bem mais intuitiva do que a do git bash

Em seguida abra sua IDE desejada (irei mostrar como importar no eclipse de exemplo)

- Abra seu eclipse selecionando a pasta em que escolheu para seu repositório
- Clique direito no explorador de projetos e selecione "import -> import"
- Dentro da pasta "General" haverá um "Existing Projects into Workspace", selecione-o
- Selecione o root directory como a pasta que clonou o repositório

O projeto irá abrir e haverá duas classes dentro de src "Principal.java" e a " Validador.java"

# Abra a Principal e execute
O resultado estará no console em baixo

Caso sua perspectiva não tenha um console, selecione:
- Window -> Show View -> Console
